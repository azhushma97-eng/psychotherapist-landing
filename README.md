# Психотерапевт — Истушкина Дарья Николаевна

  Лендинг-страница врача-психотерапевта. Целевая аудитория — молодые женщины (18–30 лет) в Беларуси с тревожными расстройствами и проблемами самоидентификации.

  ## Технологии

  - **Frontend:** React + TypeScript + Vite + Tailwind CSS + shadcn/ui
  - **Backend:** Node.js + Express
  - **База данных:** PostgreSQL + Drizzle ORM
  - **Анимации:** Framer Motion

  ## Запуск

  ```bash
  npm install
  npm run dev
  ```

  ## Секции сайта

  1. Hero — заголовок с именем и кнопками записи
  2. О враче
  3. С чем помогаю (6 направлений)
  4. Как проходит терапия (3 шага)
  5. Форматы встреч (очно / онлайн)
  6. Отзывы клиенток
  7. FAQ (5 вопросов)
  8. Форма записи на консультацию
  9. Футер
